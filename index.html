What I Do

I work with most of the popular web technologies and frameworks: Rails, Django, WordPress, JavaScript, etc. I've dabbled in a lot of stuff and love learning new languages, libraries and tools.

Recent projects have included:

  • A vaguely NaNoWriMo-inspired random motivation generator. Written as a weekend project because I wanted to learn Flask. Open source.

  • Writing a WordPress plugin that allows you to post to your blog via email. This was my project as an OPW intern for the Summer 2013 round. Open source.

  • A JavaScript-based frontend that graphed data retrieved from a backend with a REST API. Built using Rickshaw and d3.js.

  • Extracting the Rails site from a combination WordPress/Rails hybrid site, updating it to embed information from the CMS, and getting it running on a new server.

  • A WordPress site for a photography magazine, which required a shopping cart plugin, custom taxonomies and theming, and an online subscription manager + paywall.

I have a M.S. in Computer Engineering and a B.S. in Computer Science, with a minor in photography.

I've been working as a programmer for 8 years, and have worked at some neat places, including Columbia University's Center For New Media Teaching and Learning.
